Segmented Video Compression

Pranav Rathod

A project designed to compress and playback video files by segmenting frames into foreground and background regions, leveraging motion estimation, DCT, and advanced segmentation tools like Detectron2 and optical flow.

Project Overview

This project showcases a robust pipeline for video compression and playback, developed for the Multimedia Systems Design course. Below is a detailed breakdown of the key aspects:

  • Compression Pipeline: Designed a system to encode `.rgb` videos by dividing frames into foreground and background using motion estimation and advanced segmentation techniques. Macroblocks were compressed using DCT (Discrete Cosine Transform) and quantized with adaptive parameters for efficient storage.
  • Advanced Segmentation: Applied Detectron2 and optical flow to segment foreground and background regions dynamically, achieving 95% accuracy in classification. This enhanced compression by prioritizing visual quality in regions of interest.
  • Custom Video Player: Developed a real-time video player capable of decoding `.cmp` files, reconstructing frames with IDCT (Inverse Discrete Cosine Transform), and synchronizing playback at 30 FPS with audio.
  • Performance Optimization: Achieved a ~70% reduction in file size by using motion estimation (Three-Step Search) and adaptive quantization for background regions.

Technical Details

  • Technologies: Python, OpenCV, Detectron2, PyTorch, Numpy, PyAudio
  • File Formats: `.rgb` (raw video input), `.cmp` (compressed output), `.wav` (audio)
  • Segmentation Tools: Detectron2 for object detection and optical flow for motion-based analysis.
  • Playback Controls: Interactive player with support for pause, resume, step-forward, and quit functionalities.

Project Features

The video compression system integrates classical compression techniques with modern segmentation tools. By classifying macroblocks into foreground and background, the system ensures efficient storage while maintaining visual fidelity. The player provides seamless playback, with real-time decoding and synchronized audio.
