Urban Renderer

Pranav Rathod

Fall 2022

This project focuses on rendering an urban environment using WebGL, incorporating various layers like buildings, parks, water, and surface. The data for these layers is loaded from a JSON file.

Project Overview

This project involves handling user interactions through a control panel, allowing users to upload their own JSON files, adjust the view, and select different projection types. Each layer is rendered using different shader programs to achieve the desired visual effects.

Demonstration

To see the project in action, download the sample file city.json. Upload this file using the control panel on the project page to visualize the urban environment.

View Demonstration

In the demonstration, you can interact with the 3D model using your mouse:

  • Rotate: Click and drag to rotate the view around the model.
  • Zoom: Use the mouse wheel to zoom in and out.
  • Pan: Hold the right mouse button and drag to pan the view.

Setting Up the Environment

To run the project, you need to set up a local web server to serve the files. The application requires WebGL support and works best in modern browsers like Firefox or Chrome.

Here are the steps to set up the environment:

  1. Clone the project repository from GitHub.
  2. Set up a local server using tools like XAMPP, WAMP, or a simple HTTP server in Python.
  3. Ensure the browser supports WebGL and the required extensions.
